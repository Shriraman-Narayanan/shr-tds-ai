## Super Quick Deployment (5 minutes!)

### Step 1: Extract and Upload to GitHub
1. Extract the zip file `TDS_Data_Analyst_Agent_20250819_160757.zip`
2. Create new GitHub repository (name it anything you like)
3. Upload ALL extracted files to your repository
4. Commit to main branch

### Step 2: Deploy to Vercel  
1. Go to vercel.com and sign in with GitHub
2. Click "New Project" 
3. Import your GitHub repository
4. Click "Deploy" (Vercel auto-detects Next.js)
5. Wait for deployment to complete

### Step 3: Add Your API Key
1. Get OpenAI API key from https://platform.openai.com/api-keys
2. In Vercel dashboard ‚Üí Project Settings ‚Üí Environment Variables
3. Add: Name: `OPENAI_API_KEY`, Value: `your-api-key`
4. Redeploy (Vercel will prompt you)

### Step 4: Update promptfoo Config
1. Copy your Vercel URL (like https://your-app.vercel.app)
2. Edit `promptfooconfig.yaml` 
3. Replace `YOUR_VERCEL_URL` with your actual URL
4. Save the file

### Step 5: Test with promptfoo
```bash
# Install promptfoo globally
npm install -g promptfoo

# Run evaluation  
promptfoo eval

# View results
promptfoo view
```
